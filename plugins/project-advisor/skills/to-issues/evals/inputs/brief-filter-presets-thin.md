# Thin Feature Brief: Wiederkehrende Filter schneller wiederfinden

## Kontext

Das interne Operations-Team arbeitet haeufig auf derselben Personenliste mit wiederkehrenden Filterkombinationen. Heute kostet es Zeit, diese Filter immer wieder manuell neu zusammenzustellen.

## Ziel

Nutzer sollen schneller zu haeufig genutzten Filterkonfigurationen zurueckkehren koennen.

## Bereits klar

- Das bestehende Listen- und Filtererlebnis soll erweitert, nicht ersetzt werden.
- Die erste Version richtet sich an interne Nutzer.

## Noch nicht ausformuliert

- Es ist noch nicht genauer beschrieben, wie solche Filterkonfigurationen sichtbar oder organisiert werden.
- Es ist noch nicht festgelegt, welche Randfaelle in der ersten Version abgedeckt werden muessen.
